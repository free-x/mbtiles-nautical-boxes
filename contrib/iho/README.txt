* Requirements

- wget
- unzip
- python-simplejson, python-xmltodict

* Processing

$ wget https://primar.ecc.no/primar/catalogue.ctl
$ file catalogue.ctl
catalogue.ctl: Zip archive data, at least v2.0 to extract
$ unzip catalogue.ctl 
$ ./readenccatalog.py
